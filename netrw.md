## Invocation

| Description                        | Command               | Keystroke             |
| ---------------------------------- | --------------------- | --------------------- |
| `--------------------------------` | `-------------------` | `-------------------` |
| Open Netrw in current window       | `:Ex`                 |                       | 
| Open Netrw in horizontal split     | `:Sex`                |                       | 
| Open Netrw in vertical split       | `:Vex`                |                       | 

* *Can also use Lexplore plugin which is built-in to some versions of Vim*
